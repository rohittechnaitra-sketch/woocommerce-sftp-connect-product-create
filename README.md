WooCommerce SFTP Product Import & Auto Create

This plugin allows you to automatically create and update WooCommerce products by connecting your store to an SFTP server. Once installed, it adds a dedicated admin menu where you can configure your SFTP credentials, including host, username, password, port, and file path.

After setup, the plugin securely fetches product data files (XML) from your SFTP server and imports them directly into WooCommerce as products. It supports automatic product creation, updating existing products, and syncing stock and pricing based on your file structure.

To ensure continuous synchronization, the plugin includes a built-in cron job system that runs at scheduled intervals. This keeps your WooCommerce store always up to date without manual intervention.

Key Features:

Admin menu for easy SFTP configuration
Secure connection to SFTP server
Automatic product import and update
Cron job for scheduled syncing
Supports bulk product creation
Saves time by fully automating product management
